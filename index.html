SAYG Sock Pattern Generator

This is a pattern generator for Sew-As-You-Go Sock on a single bed flat bed knitting machine. No ribber required! This pattern is designed for you to use any yarn, any gauge and any foot size. This is a cuff down pattern with a 2x1 mock rib folded cuff. If you would like to use a different type of cuff, you are welcome to modify this.

You will, first, need to create a gauge swatch with your yarn of choice at your desired tension. Then, you will input your measurements into the calculator below, and it will generate a pattern for you.

This SAYG sock method is based on Diana Sullivan's sock tutorial. Please watch her video for details on this method. She suggests using a solid or variegated yarn for this project. Striping yarns won't line up correctly.

All dimensions should be input as inches.












SAYG Sock Pattern

Mock Rib Folded Cuff

  1. Cast on XX in 2x1 rib.
  2. Knit a few rows of waste yarn and ravel cord.
  3. Tension XX, MC yarn, Knit XX rows.
  4. Rehang your live stitches to create the folded cuff. Remove waste yarn.
  5. Tension XX, Knit XX rows to the heel.

Short Row Heel

  1. Set carriage to hold. Short row heel to XX center stitches. Decreasing 1 stitch each row.
  2. Short row heel back out full length of cast on. Increasing 1 stitch each row.
  3. Knit XX rows to the toe.

Short Row Toe

  1. The toe is shaped the same way as the heel. Set carriage to hold. Short row to XX center stitches. Decreasing 1 stitch each row.
  2. Short row back out full length of cast on. Increasing 1 stitch each row.

Top of Sock

  1. Knit back up the top of the sock making sure you pick up each row only one time.
  2. Complete the cuff and cast off.
